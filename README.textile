h2. GameFAQs with Shoes

A simple client for accessing information about "www.gamefaqs.com":www.gamefaqs.com using my gamefaqs "ruby library":http://github.com/Soleone/gamefaqs.


h3. Installation

# Install Shoes first if not already available
# Download this project using @git clone git://github.com/Soleone/gameshoes.git@
# Run the main file: @shoes gameshoes.rb@
